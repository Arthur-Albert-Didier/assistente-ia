# Roadmap

Evolução planejada do projeto em fases.

---

## Fase 1 — Assistente Básico

**Objetivo:** fazer a comunicação funcionar de ponta a ponta.

**Entregas:**
- [ ] Repositório configurado no GitHub
- [ ] Estrutura de pastas criada
- [ ] Bot no Telegram respondendo mensagens
- [ ] Integração com modelo de IA (API externa)
- [ ] Deploy no notebook servidor

**Resultado esperado:** enviar uma mensagem no Telegram e receber uma resposta gerada por IA.

---

## Fase 2 — Sistema de Comandos

**Objetivo:** o assistente começa a executar ações concretas.

**Entregas:**
- [ ] Banco de dados SQLite configurado
- [ ] Registro de estudos
- [ ] Registro de corridas e treinos
- [ ] Consulta de registros por período
- [ ] Identificação automática de intenção

**Resultado esperado:** registrar e consultar atividades via mensagem de texto.

---

## Fase 3 — Memória Persistente

**Objetivo:** o assistente passa a ter contexto real das conversas e da rotina.

**Entregas:**
- [ ] Histórico de conversas salvo no banco
- [ ] A IA usa o histórico para respostas mais relevantes
- [ ] Sistema de lembretes com notificações
- [ ] Relatórios semanais automáticos

**Resultado esperado:** o assistente lembra do que foi dito e do que aconteceu nos dias anteriores.

---

## Fase 4 — Automações

**Objetivo:** o assistente passa a controlar o computador e executar tarefas.

**Entregas:**
- [ ] Abertura de programas por comando
- [ ] Execução de scripts locais
- [ ] Geração de relatórios em arquivo
- [ ] Integração com outras ferramentas locais

**Resultado esperado:** dar comandos pelo Telegram e ver ações acontecerem no computador.

---

## Fase 5 — IA Avançada

**Objetivo:** o assistente começa a ser proativo e a oferecer sugestões baseadas em padrões.

**Entregas:**
- [ ] Substituição da API externa por modelo local (Ollama)
- [ ] Análise de hábitos com base no histórico
- [ ] Sugestões de estudo personalizadas
- [ ] Planejamento de rotina assistido por IA
- [ ] Relatórios com insights de desempenho

**Resultado esperado:** o assistente sugere ações, identifica padrões e ajuda a tomar decisões com base em dados reais da rotina.

---

## Visão de Longo Prazo

O objetivo final é transformar o assistente em uma **central de comando pessoal**: um sistema que centraliza toda a vida acadêmica, projetos, hábitos e tarefas — acessível de qualquer lugar, rodando em hardware próprio, com total controle dos dados.

---

## Princípios que Guiam o Desenvolvimento

- **Privacidade em primeiro lugar** — dados ficam no próprio hardware sempre que possível
- **Evolução gradual** — cada fase deve ser útil por si só
- **Aprendizado contínuo** — o projeto é também uma jornada de aprendizado em programação, IA e arquitetura de software
- **Código limpo e documentado** — o repositório serve também como portfólio
